# 💫Study Case: Sistem Penghitung Ongkos Ojek Online (Jakarta)

Implementasi dari study case praktikum OOP Java.

## Deskripsi
Program ini menghitung biaya perjalanan ojek online berdasarkan jarak tempuh.

## Struktur
- `Ojek.java`: Class OOP untuk menghitung biaya ojek
- `MainOjek.java`: Main class untuk menjalankan versi OOP
- `MainProcedural.java`: Main class untuk menjalankan versi procedural

## Cara Menjalankan
1. Compile: `javac *.java`
2. Jalankan OOP: `java MainOjek`
3. Jalankan Procedural: `java MainProcedural`
